# Colonize And Conquer

## Technology Stack
  - Django
  - Python
  - Vue.js
  - JavaScript
  - HTML
  - CSS
  - FontAwesome
  - Bootstrap

### Installation

Install the dependencies and start the server.

```sh
$ cd colonize-and-conquer
$ pip install -r requirements.txt
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver
```
